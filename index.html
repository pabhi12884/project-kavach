Project Kavach

An AI-powered agent that defends your Gmail inbox from phishing, malware, and AI threats.

The Problem

Email threats are evolving. Attackers use AI to craft phishing messages that bypass traditional spam filters. Malware hides in attachments and links appear legitimate.

Our Solution

Project Kavach scans each email in your inbox using advanced techniques: file hash checks, AI-content detection, header analysis, and sandbox execution — giving you clear explanations and risk scores.

Explainable Risk Score

Every email gets a detailed risk score based on over 10 cybersecurity signals.

AI Text Detection

Detects emails written by AI to identify impersonation and social engineering.

File Sandboxing

Suspicious attachments are tested in a virtual sandbox (e.g. Any.run) automatically.

Gmail Integration

Integrated directly into your inbox using Gmail API for real-time analysis.

Why Project Kavach?

We don't just block threats. We explain them. Kavach learns from user feedback and improves continuously — with transparency and trust at the core.

How It Works

  1. Connect your Gmail securely with OAuth
  2. Every new email is scanned in real time
  3. Each threat is explained, and a score is shown
  4. Users can submit feedback to improve the agent
