Project Kavach – Email Security AI-Agent

Why This Matters: Email remains the most common entry point for cyberattacks. Phishing, spoofing, and malware attachments often bypass basic filters. Evolving social engineering tactics require smarter, adaptive security solutions.

What Project Kavach Does: An open-source AI agent that deeply analyzes every Gmail email with trusted methods — no black-box decisions.

  • Inspects headers and content for phishing, spoofing, and impersonation
  • Validates domains and URLs using real-time threat intelligence
  • Analyzes attachments securely via sandboxing (e.g., Any.Run)
  • Explains why emails are marked spam or moved to trash
  • Allows user corrections to help the agent learn and improve
  • Designed with security-first principles — no persistent storage risks

How It Evolves: User feedback enables contextual adaptation and transparency.

Our Vision: Project Kavach is fully open-source, delivering real solutions for real problems through community collaboration.

Currently focusing on Gmail integration to test and improve, aiming for a smarter, safer inbox.
