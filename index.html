Project Kavach – Secure Email Agent

Why This Matters: Email is the most common entry point for cyberattacks. Phishing, spoofing, and malware-laced attachments often bypass traditional filters. With the rise of targeted social engineering, basic spam filters aren’t enough. We need a smarter, adaptive solution.

What Project Kavach Does: An open-source email agent designed to provide intelligent, explainable security for Gmail. The agent deeply inspects every email using trusted techniques — no shortcuts or black-box decisions.

  • Scans headers and body for phishing, spoofing, and impersonation attempts
  • Validates domains and URLs with real-time threat intelligence
  • Analyzes attachments using isolated sandboxes
  • Explains why emails are marked spam or sent to trash
  • Lets users correct decisions, learning and adapting over time
  • Built with security-first principles no persistent storage or remote risk and taking measures to prevent abuse, hijacking, or misclassifications

How It Evolves: User feedback helps the agent improve, building contextual understanding tailored to your inbox — with transparency and safety.

Our Vision: Project Kavach is fully open-source. It’s a real solution to a real problem, created by and for the security community.

We’re starting with Gmail integration to test and improve the model, then expanding. Join us to build a smarter, safer inbox.
